## Branch Naming Convention 
Always include your name: - sara/homepage - ravi/menu-page - priya/deployment-docs 
Format: yourname/what-you-are-building 
## How We Work — Step by Step 
1. Always create a branch from main before starting 
2. Do all your work on your branch 
3. Sync branch with main before opening PR 
4. Open a Pull Request when done 
5. Get at least 2 reviews and approvals 
6. Only then merge into main
