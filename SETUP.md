## Branch Naming Convention 
Always include your name: - sara/homepage - ravi/menu-page - priya/deployment-docs 
Format: yourname/what-you-are-building 
## How We Work 
### Feature PRs - Needs 2 approvals before merging 
### Documentation PRs - Needs 2 approvals — shared docs affect whole team 
### Hotfix PRs - Urgent fixes — 1 approval is enough for speed
